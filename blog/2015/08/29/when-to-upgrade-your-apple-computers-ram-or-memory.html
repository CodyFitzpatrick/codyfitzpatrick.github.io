When to Upgrade Your Apple Computer’s RAM or Memory

How do you know if you should upgrade your Apple computer’s RAM? If you use your computer and it has hiccups when performing routine tasks, like opening a document or using an application, you might need more memory. How can you tell if this is the case?

On Mac, open up Activity Monitor. When doing what you normally do (like browsing the web or working with documents), view your Memory usage by viewing the “Memory Used” in comparison to “Physical Memory.” If your Memory Used is high, or almost exactly the same as your Physical Memory like in the picture below, your computer needs more RAM!
